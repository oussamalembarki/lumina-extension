# Lumina - Focus Mode for YouTube

A Chrome extension that transforms YouTube into a distraction-free learning environment.

Lumina helps students and developers stay focused by hiding distracting elements and providing a built-in note-taking system while watching videos.

## Features

- Hide YouTube distractions (recommendations, comments, Shorts)
- Floating notepad with automatic local saving
- Add timestamps while taking notes
- Focus tracking during study sessions
- Privacy-first: no accounts, no data collection

## Tech Stack

- JavaScript
- Chrome Extension Manifest V3
- Chrome Storage API
- HTML / CSS

## Installation

1. Clone this repository
2. Open `chrome://extensions/`
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
